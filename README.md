# 扇贝桌面版 🐚 
学习使用 Electron 和 Vue.js 的练习项目 🎉。利用扇贝 API 开发的跨平台桌面应用 💻。

# 功能
  * 单词查询
  * 系统托盘快速查询
  * 用户登录后可以从主界面或托盘窗口添加到生词本
